# StatArb

> Statistical Arbitrage Project

这是一个统计套利(Stat Arb)的项目，主要用于：

- 期货同品种跨期套利
- 期货跨品种套利

## 项目的主要文件结构为

    StatArb/
    |-- R/         ## 主要存放 R 代码与函数
    |-- scr/       ## py, cpp
    |-- data/      ## 原始数据，不可改动
    |-- doc/       ## 项目文档说明
    |-- fig/       ## 产生的图形
    |-- output/    ## 由脚本生成的数据、文件等
    |-- main.R     ## 主要的执行函数
